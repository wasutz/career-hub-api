## Career hub
RESTful API for job posting.

## Feature
- User authentication with JWT
- CRUD operations for job postings
- Pagination support
- Swagger API documentation

## Getting Started

### Local development

```bash
$ docker-compose up
```

### Run tests

```bash
$ npm run test
```

### Tech Stack
- NestJS
- PostgreSQL
- TypeORM
- Docker & Docker Compose
- Swagger (OpenAPI)
